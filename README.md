# ocservBook
версия 0.01

 Руководства , инструкции и компиляции из разных источников
  для развертывания и управления сервером Open Connect на
  платформе  виртуальной машины с Debian 12

 Список включает в себя :
 1. pdf Man руководство на русском языке утилиты управления сертификатами certtool;
 2. doc руководство по созданию виртуальной машины с ОС Debian в вреде виртуализации VirtualBox;
 3. doc руководство по уствновке и настройке сервера OpenConnect v.1.3;
 4. doc руководство по созданию сертификатов;
 5. doc руководство по установке клиента Open Connect на машине с ОС Windows 10;
 6. PDF схема подключения к серверу.

 7. Дополнительно отдельный матриал по настройке в среде GNS3^
 Публикую предварительную версию  книги о развёртывании OpenConnect Server в лабораторной среде.
Это не “быстрый гайд за вечер” и не пересказ документации.
Книга задумывается как практическое учебное руководство для тех, кто хочет понять, что именно происходит в системе и сети, а не просто повторить команды.
В этих главах разобраны:
подготовка среды виртуализации (VMware + GNS3);
корректная установка Debian 12 из ISO, без cloud-образов и скрытых ловушек;
базовая сетевая схема, пригодная для дальнейшей работы с VPN;
контрольные точки, позволяющие понять, что всё сделано правильно и можно идти дальше;
обновление системы и установка всех необходимых пакетов для сборки и компиляции сервера OCserv;
Получение, сборка и компиляция сервера OCserv;
Генерация сертификатов (PKI);
Конфигурация файла /etc/ocserv/ocserv.confg;
Настройка аутентификации;
Тестирование подключения клиента в Windows.
Материал ориентирован на:
начинающих администраторов;
инженеров, которые только осваивают GNS3 и Linux;
тех, кто устал от инструкций в стиле «сделайте так, у вас должно заработать».
⚠️ Важно:
Это ранний доступ. Структура уже выстроена, логика зафиксирована, но текст продолжает редактироваться и дополняться. Обратная связь от читателей будет учтена в финальной версии книги.
Если вам важно не просто “поднять VPN”, а разобраться, почему он работает именно так — этот материал для вас.
